# 🦾 Роборука - Установка приложения на Android

## 📱 Способ 1: Веб-приложение (PWA) - РЕКОМЕНДУЕТСЯ

Это приложение работает как настоящее - устанавливается на главный экран и работает без браузера.

### Шаг 1: Открой приложение на телефоне

**Вариант A - Через WiFi (телефон и ПК в одной сети):**

На ПК запусти сервер:
```bash
cd /home/tarixxk/industrial_ai_hub
python3 -m http.server 8080
```

На телефоне открой браузер и перейди:
```
http://192.168.0.103:8080/android/app.html
```
(замени 192.168.0.103 на IP твоего ПК)

**Вариант B - Загрузить на хостинг:**
1. Загрузи папку `android/` на любой хостинг (GitHub Pages, Netlify, Vercel)
2. Открой ссылку на телефоне

### Шаг 2: Установи как приложение

В браузере Chrome на телефоне:
1. Открой страницу приложения
2. Нажми **⋮ (три точки)** в углу
3. Выбери **"Установить приложение"** или **"Добавить на гл. экран"**
4. Подтверди установку

✅ Готово! Иконка "Роборука" появится на главном экране.

### Шаг 3: Запусти приложение

1. Найди иконку 🦾 Роборука на главном экране
2. Открой (работает без браузера!)
3. Разреши доступ к камере
4. Покажи руку в камеру

## 📲 Способ 2: APK через Website 2 APK

Если нужен именно APK файл:

### Вариант A: Online конвертер
1. Зайди на https://website2apk.com/
2. Загрузи файл `app.html` (или укажи URL)
3. Скачай готовый APK
4. Установи на телефон

### Вариант B: AppsGeyser
1. Зайди на https://www.appsgeyser.com/
2. Выбери "Website" шаблон
3. Укажи URL или загрузи HTML
4. Скачай APK

### Вариант C: Web2Apk (приложение)
1. Установи "Web2Apk" из Play Store
2. Открой app.html через приложение
3. Создай APK

## 🔌 Подключение к Arduino

### Через Bluetooth (HC-05/HC-06):

**В приложении:**
1. Нажми 📡 (кнопка подключения)
2. Выбери устройство HC-05
3. Подтверди сопряжение (пароль: 1234 или 0000)

**Схема подключения:**
```
Arduino Nano:
- TX (D1) → RX HC-05
- RX (D0) → TX HC-05  
- 5V → VCC HC-05
- GND → GND HC-05
```

### Через WiFi (ESP32):

Нужно прошить ESP32 прошивкой для приёма команд.

## 🛠️ Решение проблем

**"Не работает камера":**
- Дай разрешение в настройках Android
- Проверь что камера не используется другим приложением

**"Рука не распознаётся":**
- Хорошее освещение
- Покажи ладонь полностью
- Держи руку на расстоянии 20-50 см

**"Не подключается Bluetooth":**
- Включи Bluetooth и GPS на телефоне
- Проверь что HC-05 запитан и мигает
- Расстояние < 10 метров

**"Приложение не устанавливается":**
- Включи "Неизвестные источники" в настройках
- Для PWA: используй Chrome браузер
