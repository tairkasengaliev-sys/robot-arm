## 🔌 Схема подключения

```
                    ┌─────────────────┐
                    │   Arduino Nano  │
                    │                 │
        ┌───────────┤  D3   → SG90 #1 │──→ Большой палец
        │           │  D5   → SG90 #2 │──→ Указательный
        │           │  D6   → SG90 #3 │──→ Средний
        │           │  D9   → SG90 #4 │──→ Безымянный
        │           │  D10  → SG90 #5 │──→ Мизинец
        │           │  D11  → SG90 #6 │──→ Запястье
        │           │  A0   → SG90 #7 │──→ Локоть
        │           │  A1   → SG90 #8 │──→ Плечо
        │           │                 │
        │           │  D0    ← HC-05  │──→ Bluetooth RX
        │           │  D1    → HC-05  │──→ Bluetooth TX
        │           │  5V    → VCC    │──→ Питание серво
        │           │  GND   → GND    │──→ Общий минус
        │           └─────────────────┘
        │
        │    ┌─────────────────────────┐
        │    │   Блок питания 5V 2A    │
        │    │   (для сервоприводов)   │
        │    └─────────────────────────┘
        │
        └────→ ВСЕ GND соединить вместе!
```

### ⚠️ ВАЖНО:
- **НЕ питай сервоприводы от Arduino!** Сгорит!
- Используй внешний БП 5V 2A+
- **Общий GND** обязателен между Arduino, БП и серво

## 🛠️ Пошаговая сборка

### Шаг 1: Сервоприводы

```
Сервопривод SG90:
    ┌─────┐
    │ 🎛️  │ ← Корпус
    │     │
    └──┬──┘
       │
    ┌──┴──┐
    │ 📌  │ ← Рычаг (вал)
    └─────┘

Провода:
🟤 Коричневый  → GND
🔴 Красный     → VCC (5V)
🟡 Жёлтый/Оранжевый → Signal (к Arduino)
```

### Шаг 2: Сборка пальцев

```
Каждый палец = 1 сервопривод + 2 фаланги

    ┌──────┐
    │ SG90 │
    └──┬───┘
       │
    ┌──┴───┐
    │ 🦴   │ ← Фаланга 1 (3-4 см)
    └──┬───┘
       │
    ┌──┴───┐
    │ 🦴   │ ← Фаланга 2 (2-3 см)
    └──────┘
```

        🦾
       ╱│╲

     ╱  │  ╲

        │

    └───────┘

## 💻 Прошивка Arduino

### robot_arm.ino (бюджетная версия)

```cpp
/*
 * Бюджетная роборука на 8 сервоприводах SG90
 * Управление через Bluetooth (HC-05)
 */

#include <Servo.h>

Servo servos[8];

// Пины сервоприводов
const int servoPins[8] = {3, 5, 6, 9, 10, 11, A0, A1};

// Текущие позиции
int positions[8] = {90, 90, 90, 90, 90, 90, 90, 90};

void setup() {
  Serial.begin(9600);
  
  // Инициализация сервоприводов
  for (int i = 0; i < 8; i++) {
    servos[i].attach(servoPins[i]);
    servos[i].write(positions[i]);
  }
  
  Serial.println("READY");
}

void loop() {
  if (Serial.available() > 0) {
    String cmd = Serial.readStringUntil('\n');
    cmd.trim();
    
    if (cmd.startsWith("SET:")) {
      parseSetCommand(cmd.substring(4));
      Serial.println("OK");
    }
    else if (cmd == "HOME") {
      moveTo(90, 90, 90, 90, 90, 90, 90, 90);
      Serial.println("HOME");
    }
    else if (cmd == "GRIP") {
      moveTo(45, 30, 30, 30, 30, 90, 120, 90);
      Serial.println("GRIP");
    }
    else if (cmd == "OPEN") {
      moveTo(90, 90, 90, 90, 90, 90, 90, 90);
      Serial.println("OPEN");
    }
  }
}

void parseSetCommand(String data) {
  int values[8];
  int idx = 0;
  int start = 0;
  
  for (int i = 0; i <= data.length() && idx < 8; i++) {
    if (i == data.length() || data.charAt(i) == ',') {
      values[idx++] = constrain(data.substring(start, i).toInt(), 0, 180);
      start = i + 1;
    }
  }
  
  if (idx == 8) {
    moveTo(values[0], values[1], values[2], values[3],
           values[4], values[5], values[6], values[7]);
  }
}

void moveTo(int p0, int p1, int p2, int p3, int p4, int p5, int p6, int p7) {
  int targets[8] = {p0, p1, p2, p3, p4, p5, p6, p7};
  
  // Плавное движение
  for (int step = 0; step < 20; step++) {
    for (int i = 0; i < 8; i++) {
      int diff = targets[i] - positions[i];
      if (abs(diff) > 1) {
        positions[i] += diff / 20;
        servos[i].write(positions[i]);
      }
    }
    delay(15);
  }
  
  // Финальная позиция
  for (int i = 0; i < 8; i++) {
    positions[i] = targets[i];
    servos[i].write(positions[i]);
  }
  delay(100);
}
```

## 📱 Подключение телефона

### Пароль Bluetooth HC-05:
- **1234** или **0000**

### Настройка HC-05:
```
HC-05      Arduino Nano
──────     ────────────
VCC   →    5V
GND   →    GND
TX    →    RX (D0)
RX    →    TX (D1)
```

## 🐛 Частые проблемы

| Проблема | Решение |
|----------|---------|
| Серво дёргаются | Добавить конденсатор 470uF на питание |
| Arduino перезагружается | Слабый БП, нужен 2A+ |
| Не все серво работают | Проверь общий GND |
| Bluetooth не подключается | Пароль 1234, TX/RX перепутаны |
| Серво греется | Нормально для SG90 |
