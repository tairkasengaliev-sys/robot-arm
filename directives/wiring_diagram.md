## 🔌 Сервопривод SG90 - Распиновка

```
        ┌─────────────┐
        │   SG90      │
        │  Сервопривод│
        │             │
        └──────┬──────┘
               │
         ┌─────┴─────┐
         │  🎛️ ВАЛ  │  ← Вращается 0-180°
         └─────┬─────┘
               │
         ┌─────┴─────┐
         │  📌 РЫЧАГ │  ← Крепится на вал
         └───────────┘
               │
        ┌──────┴──────┐
        │             │
    ┌───┴───┐     ┌───┴───┐
    │ 🦴    │     │ 🦴    │  ← Фаланги (3D печать/картон)
    └───────┘     └───────┘


        ПРОВОДА (3 шт, ~15см):
        
        🟤 Коричневый  ──→ GND (минус)
        🔴 Красный     ──→ VCC (5V, плюс)
        🟡 Жёлтый      ──→ SIG (сигнал, к пину Arduino)
        
        Или:
        
        ⚫ Чёрный      ──→ GND
        🔴 Красный     ──→ VCC
        🟠 Оранжевый   ──→ SIG
```

### Подключение к Shield V5.0:

```
Серво (3 пина)         Sensor Shield V5.0
────────────           ──────────────────
🟤 GND  ───────────────→ GND (любой на той же линии)
🔴 VCC ────────────────→ VCC (любой на той же линии)
🟡 SIG ────────────────→ Пин серво (D3, D5, D6, D9, D10, D11, A0, A1)
```

## 🔵 Bluetooth HC-05 - Схема

```
        ┌──────────────────────┐
        │   HC-05 Bluetooth    │
        │   ┌──────────────┐   │
        │   │   АНТЕННА    │   │  ← Маленький проводок
        │   └──────────────┘   │
        │                      │
        │  📌 VCC ──→ 5V       │
        │  📌 GND ──→ GND      │
        │  📌 TX  ──→ D0 (RX)  │
        │  📌 RX  ──→ D1 (TX)  │
        │  📌 KEY ──→ NC       │  ← Не подключать
        └──────────────────────┘

        Размеры: ~27×13×3 мм
        
        Пароль по умолчанию: 1234 или 0000
        Baud rate: 9600
```

### Подключение HC-05 к Shield:

```
HC-05              Sensor Shield V5.0 (Digital секция)
─────              ───────────────────────────────────
VCC ───────────────→ 5V (пин рядом с D13)
GND ───────────────→ GND (рядом с 5V)
TX  ───────────────→ D0 (RX Arduino)
RX ────────────────→ D1 (TX Arduino)
```

## 🔧 Проверка правильности подключения

### Чек-лист:

```
□ Все 8 серво подключены к правильным пинам
□ GND всех серво соединены с GND на Shield
□ VCC всех серво соединены с VCC на Shield
□ GND на Shield соединён с GND Arduino
□ HC-05 подключён к D0/D1
□ БП 5V подключён к VIN на Shield
□ USB подключён к Arduino
□ Все провода надёжно вставлены
□ Нет коротких замыканий
```

### Первый запуск:

```bash
# 1. Проверь схему
# 2. Подключи USB к Arduino
# 3. Загрузи прошивку
# 4. Включи БП 5V
# 5. Открой Serial Monitor (9600)
# 6. Отправь: STATUS
# 7. Ответ: POS:90,90,90,90,90,90,90,90
```

### Если не работает:

```
❌ Серво дёргаются:
   → Слабый БП, нужен 3A минимум
   → Добавь конденсатор 470uF

❌ Arduino перезагружается:
   → Серво питаются от Arduino!
   → Раздели питание

❌ HC-05 не отвечает:
   → Проверь TX/RX (перепутаны?)
   → Пароль 1234 или 0000

❌ Одно серво не работает:
   → Проверь подключение (3 пина)
   → Может бракованное
```

## 🎯 Итоговая схема (кратко)

```
БП 5V 3A ──→ Shield V5.0 VIN ──→ 8× Серво

D3  → #0 Большой

D11 → #5 Запястье
```
