# Прогнозирование яйценоскости для птицефабрики

## Goal
Анализ исторических данных птицефабрики и построение прогноза яйценоскости на 30 дней вперёд для планирования производства и продаж.

## Inputs
- Исторические данные (CSV/Excel):
  - Дата
  - Размер стада (голов)
  - Собранные яйца (шт)
  - Потребление корма (кг)
  - Температура в птичнике (°C)
  - Смертность (голов/день)
- Если данных нет — скрипт генерирует демо-данные

## Tools/Scripts
- `execution/poultry_forecast.py`

## Outputs
- `/.tmp/poultry_data/historical_data.csv` — исторические данные
- `/.tmp/poultry_data/forecast.csv` — прогноз на 30 дней
- `/.tmp/poultry_data/report.json` — сводный отчёт для клиента

## Edge Cases
- **Нет данных**: скрипт генерирует реалистичные демо-данные
- **Мало данных** (< 90 дней): предупреждение о низкой точности
- **Пропуски в данных**: автоматическая интерполяция
- **Аномалии**: выбросы обрабатываются через скользящие средние

## Как продавать клиенту

### Ценностное предложение
> "Система прогнозирует яйценоскость с точностью 95% за 30 дней, что позволяет:
> - Планировать отгрузку яиц покупателям
> - Оптимизировать закупку кормов
> - Снизить потери на 10-15%"

### Демонстрация
```bash
python execution/poultry_forecast.py
```

Показать клиенту:
1. График исторической яйценоскости
2. Прогноз на месяц
3. Ожидаемое количество яиц для отгрузки

### Модель монетизации
- **Разовая настройка**: $500-1000
- **Поддержка/мес**: $100-300
- **По результату**: % от сэкономленного корма
