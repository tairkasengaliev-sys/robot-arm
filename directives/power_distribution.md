# ⚡ Правильное питание роборуки

## ❌ НЕПРАВИЛЬНО (сгорит!)

```
БП 5V → Arduino 5V → Сервоприводы
         ⬆️
    Регулятор Arduino (500mA макс)
    ❌ 8 серво × 500mA = 4A → СГОРИТ!
```

---

## ✅ ПРАВИЛЬНО (раздельное питание)

```
                    ┌──────────────┐
                    │  БП 5V 3A+   │
                    │  (для серво) │
                    └──────┬───────┘
                           │
         ┌─────────────────┼─────────────────┐
         │                 │                 │
    ┌────┴────┐       ┌────┴────┐       ┌────┴────┐
    │ SG90 #1 │       │ SG90 #2 │       │ ... #8  │
    └────┬────┘       └────┬────┘       └────┬────┘
         │                 │                 │
         └─────────────────┴─────────────────┘
                           │
                      ┌────┴────┐
                      │  GND    │──────┐
                      └─────────┘      │
                                       │
                    ┌──────────────────┘
                    │
         ┌──────────┴──────────┐
         │   Arduino Nano      │
         │   (отдельное питание)│
         └─────────────────────┘
```

## 🔌 Схема подключения (детально)

### Вариант 1: Два блока питания

```
БП 5V 3A                БП 9V 1A (или USB)
──────                  ──────
  + ───→ VCC всех серво    + ───→ VIN Arduino
  - ───→ GND всех серво    - ───→ GND Arduino
           │                        │
           └──────────┬─────────────┘
                      │
                 ВСЕ GND ВМЕСТЕ!
```

### Вариант 2: Один БП + стабилизатор

```
БП 12V 3A
───────
  + ───→ LM2596 (Step-down) ───→ 5V для серво
  - ───→ GND всех серво + GND Arduino
  + ───→ VIN Arduino (через диод)
```

### Вариант 3: USB + отдельный БП для серво

```
USB (5V) ───→ Arduino (для логики)
БП 5V 3A ───→ Сервоприводы
GND ─────────→ Общий
```

## ⚡ Потребление сервоприводов SG90

| Состояние | Ток на 1 серво | Ток на 8 серво |
|-----------|----------------|----------------|
| Без нагрузки | 100-200 mA | 800-1600 mA |
| Среднее | 300-400 mA | 2400-3200 mA |
| Под нагрузкой | 500-700 mA | 4000-5600 mA |
| Stall (упор) | 800+ mA | 6400+ mA |

**Рекомендация:** БП **5V 3A минимум**, лучше **5A**

## 🛡️ Защита

### 1. Конденсатор на питание серво

```
БП 5V ──┬──→ VCC серво
        │
     ┌──┴──┐
     │ 470 │ ← Электролитический конденсатор
     │ uF  │   (минимум 10V)
     └──┬──┘
        │
GND ────┴──→ GND серво
```

**Зачем:** Сглаживает скачки тока при движении

### 2. Диод на питание Arduino

```
БП 9V ──┬──→|──→ VIN Arduino
        │   1N4007
        │
GND ────┴──→ GND Arduino
```

**Зачем:** Защита от обратного тока

### 3. Предохранитель (опционально)

```
БП 5V ──┬──→ Fuse 3A ──→ Серво
        │
GND ────┴──→ GND
```

## 📐 Готовая плата распределения питания

Сделай сам:

```
          ┌─────────────────────────────┐
          │   Плата распределения 5V    │
          │                             │
БП 5V ──→ │ ● ● ● ● ● ● ● ● ● ● ● ● ●  │ ──→ 8× Сервоприводы
          │ + + + + + + + + + + + + + + │     (VCC)
GND ────→ │ - - - - - - - - - - - - - - │ ──→ GND серво + Arduino
          │                             │
          │        [470uF 10V]          │ ← Конденсатор
          └─────────────────────────────┘
```

Материалы:
- Текстолит или макетная плата
- Винтовые клеммы 5mm
- Конденсатор 470uF 10V+

## 🧪 Тестирование

### Мультиметром:

1. **Проверь напряжение** на выходе БП: должно быть 5.0±0.2V
2. **Проверь под нагрузкой**: подключи 4 серво, должно быть 4.8-5.2V
3. **Проверь пульсации**: AC режим, должно быть < 100mV

### Без мультиметра:

```cpp
// Тест: двигай все серво одновременно
void stressTest() {
  for (int i = 0; i < 8; i++) {
    servos[i].attach(servoPins[i]);
  }
  
  while (true) {
    // Все в 0°
    for (int i = 0; i < 8; i++) servos[i].write(0);
    delay(1000);
    
    // Все в 180°
    for (int i = 0; i < 8; i++) servos[i].write(180);
    delay(1000);
    
    // Все в 90°
    for (int i = 0; i < 8; i++) servos[i].write(90);
    delay(1000);
  }
}
```

**Наблюдай:**
- ✅ Серво двигаются плавно — ОК
- ⚠️ Arduino перезагружается — слабый БП
- 🔥 Arduino греется — сгорит скоро!
- 💨 Дым — уже сгорел 😅

## 🎯 Итоговая схема

```
┌─────────────────────────────────────────────────────────┐
│                    РОБОРУКА ПИТАНИЕ                     │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   СЕТЬ 220V ──→ [БП 5V 3A] ──┬─→ Сервоприводы (8 шт)   │
│                              │   VCC: красный          │
│                              │   GND: коричневый       │
│                              │                         │
│                              ├─→ [470uF] ─→ GND        │
│                              │                         │
│                              └─→ GND ──┬─→ Arduino GND │
│                                        │               │
│   USB ──→ Arduino Nano ────────────────┘               │
│   (5V от USB для логики)                               │
│                                                         │
│   Signal: Arduino пины → Серво (жёлтый)                │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**Главное правило:** 
> ⚡ **СИЛОВАЯ ЧАСТЬ (серво) ≠ ЛОГИКА (Arduino)**
> 
> Раздельное питание, общий GND!
