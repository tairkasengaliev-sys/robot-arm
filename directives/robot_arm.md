# Роборука на Arduino с управлением через камеру

## 📋 Компоненты

| Компонент | Количество | Примечание |
|-----------|------------|------------|
| Arduino Uno/Nano | 1 | Микроконтроллер |
| Сервопривод SG90 | 8 | 9g микро серво |
| Питание 5V 2A | 1 | Для сервоприводов |
| Макетная плата | 1 | Для соединений |
| Перемычки | 20+ | M-M, M-F |

## 🔌 Схема подключения

```
Arduino          Сервоприводы
======           ============
Pin 3   ────────  Серво 1: Большой палец
Pin 5   ────────  Серво 2: Указательный
Pin 6   ────────  Серво 3: Средний
Pin 9   ────────  Серво 4: Безымянный
Pin 10  ────────  Серво 5: Мизинец
Pin 11  ────────  Серво 6: Запястье
Pin A0  ────────  Серво 7: Локоть
Pin A1  ────────  Серво 8: Плечо

GND     ────────  Общий минус (ВСЕХ серво!)
5V      ────────  ПИТАНИЕ ОТ ВНЕШНЕГО БП 5V!
```

⚠️ **ВАЖНО:** Не питайте сервоприводы от Arduino! Используйте внешний БП 5V 2A+.

## 📦 Установка

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск управления через камеру
python execution/robot_arm_control.py

# ИЛИ веб-интерфейс
streamlit run app_arm.py
```

## 🎮 Использование

### Режим 1: Прямое управление (камера)
```bash
python execution/robot_arm_control.py
```

**Управление:**
- Покажите руку в камеру → роборука повторяет
- `H` → домашняя позиция
- `G` → захват
- `O` → открыть ладонь
- `Q` → выход

### Режим 2: Веб-интерфейс
```bash
streamlit run app_arm.py
```

Откройте http://localhost:8501

## 🔧 Загрузка прошивки

1. Откройте `arduino/robot_arm.ino` в Arduino IDE
2. Выберите плату: Arduino Uno
3. Выберите порт: /dev/ttyUSB0 или COM3
4. Нажмите "Загрузить"

## 🎯 Как это работает

```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│   Камера    │ ──→ │    Python    │ ──→ │   Arduino   │
│  (веб-камера)│     │ MediaPipe    │     │  (сервоприводы)│
└─────────────┘     │ OpenCV       │     └─────────────┘
                    │ трекинг руки │
                    └──────────────┘
```

1. **Камера** захватывает видео 30 FPS
2. **MediaPipe** определяет 21 точку руки
3. **Python** вычисляет углы суставов
4. **Arduino** двигает сервоприводы

## 🛠️ Настройка

### Если камера не работает:
```bash
# Проверьте доступные камеры
ls /dev/video*  # Linux
# или
python -c "import cv2; print(cv2.getBuildInformation())"
```

### Если Arduino не подключается:
```bash
# Проверьте порт
ls /dev/ttyUSB*  # Linux
# или
python -m serial.tools.list_ports  # Все порты
```

### Калибровка углов:
В `execution/robot_arm_control.py` измените:
- `MIN_ANGLE` / `MAX_ANGLE` - границы
- Коэффициенты в `calculate_angles_from_hand()`

## 📹 Демонстрация

1. Запустите `python execution/robot_arm_control.py`
2. Покажите ладонь в камеру
3. Сжимайте/разжимайте пальцы
4. Роборука повторяет движения
